# AI阅读助手浏览器插件

一款基于 **AI Agent 架构** 的浏览器阅读助手插件，可在任意网页上选中文本，通过 AI 实现即时解释、专业翻译、面试考点分析和全文摘要，致力于提升前端开发者的文档阅读效率。

## 项目亮点

- 面向前端开发者的阅读场景，聚焦技术文档理解与知识沉淀
- 划词即用，无需切换页面，减少阅读上下文中断
- 基于 Agent 分层设计，可扩展、可维护、可持续迭代
- 支持流式响应与高亮持久化，提升交互体验与信息留存

## 核心功能

- 📖 **划词解释**：选中技术文本，AI 用通俗语言解释概念与上下文
- 🌐 **划词翻译**：选中英文内容进行专业翻译，保留代码和变量名不做改动
- 🎯 **面试考点**：识别是否为前端高频考点，并自动生成面试题与回答方向
- 💬 **追问对话**：可基于解释/翻译结果继续追问，逐步深入理解
- 📄 **全文摘要**：一键提取网页正文并生成三句话总结
- 🖍️ **划词高亮**：提供五色高亮标记重点，支持跨会话持久化
- 📋 **划词历史**：自动记录查询历史，支持点击回溯与复用
- ⚡ **流式输出**：基于 SSE 的流式响应，呈现逐字打字机效果

## 技术栈

- **前端框架**：React 18
- **语言与工程化**：TypeScript
- **浏览器扩展框架**：WXT
- **大模型服务**：智谱 GLM-4-Flash
- **正文提取**：Mozilla Readability

## AI Agent 架构设计

本项目采用分层 Agent 设计，将浏览器阅读过程拆解为“感知-记忆-决策-执行”四个阶段：

- **感知层（Perception）**
  - 监听鼠标划选事件与交互行为
  - 利用 Readability 提取网页正文，构建可处理语料
- **记忆层（Memory）**
  - 存储当前网页全文上下文
  - 管理对话历史、划词记录与高亮状态
- **决策层（Decision）**
  - 识别用户意图（解释、翻译、考点、摘要、追问等）
  - 自动选择对应 Prompt 模板与调用策略
- **执行层（Execution）**
  - 调用大模型 API 生成结果
  - 采用 SSE 流式渲染，提供实时反馈体验

## 快速开始

### 1) 克隆项目

```bash
git clone <your-repo-url>
cd ai-reader
```

### 2) 安装依赖

```bash
npm install
```

### 3) 启动开发

```bash
npm run dev
```

### 4) 加载浏览器扩展

在 Edge/Chrome 中打开“扩展管理”页面，启用开发者模式后，加载以下目录：

```text
.output/chrome-mv3
```
## 📥 手动安装

1. 打开 [Releases](https://github.com/Nonoxs111/ai-reader/releases) 页面
2. 下载最新的 `ai-reader.zip`
3. 解压到你电脑上的任意文件夹
4. 打开浏览器，地址栏输入 `edge://extensions`（Chrome 输入 `chrome://extensions`）
5. 开启左下角 **"开发人员模式"**
6. 点击 **"加载解压缩的扩展"**
7. 选择刚才解压的文件夹
8. 插件图标会出现在浏览器工具栏右上角，点击即可使用
